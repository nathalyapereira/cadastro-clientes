## Principais Tecnologias
- Angular 20;
- PrimeNG 20;
- TailwindCSS;
- RxJS;
- ngx-mask;

## Funcionalidades Implementadas
- Formulários implementados com Reactive Forms, FormBuilder e Validators;
- Validações dinâmicas usando valueChanges;
- Máscaras aplicadas com p-inputMask e ngx-mask;
- Validação customizada para o campo de CPF com diretiva própria;
- Formatação customizada para contato (fixo e celular) utilizando Pipe;
- Gerenciando com serviço + BehaviorSubject 
- Componentes reutilizáveis.
